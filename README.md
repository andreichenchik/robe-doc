# ROBE

ROBE is a mobile application for digital wardrobe management. The app helps users create a digital copy of their real wardrobe, organize clothing items, and compose outfits.

## Core Capabilities

- **Wardrobe management** — add, edit, and organize clothing items.
- **AI recognition** — automatic detection of clothing type, category, and attributes from a photo.
- **Outfit creation** — combine items into outfits using an interactive canvas.
- **Filtering & search** — find items by category, type, color, and tags.
